# Smart Study Planner (Intelligent Agent)

## 📌 Overview
This project is an AI-based intelligent agent that analyzes student performance and generates a personalized weekly study plan.

## 🧠 Concepts Used
- Intelligent Agents (Goal-Based Agent)
- Decision Making
- Rule-Based System

## ⚙️ Features
- User inputs number of subjects
- Takes performance rating (out of 5)
- Classifies subjects (Strong / Average / Weak)
- Generates study strategy
- Creates weekly study plan

## ▶️ How to Run
1. Install Python
2. Run:
   python main.py
